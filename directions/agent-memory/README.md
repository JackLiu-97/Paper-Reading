# Agent Memory

本方向收录 Agent 记忆相关论文和资料，重点关注短期记忆、长期记忆、检索增强记忆、情景记忆、记忆压缩、遗忘机制和多轮任务状态管理。该方向可服务于自动化评审中的长期上下文、评审历史、审稿偏好和多轮审稿流程建模。
